# YC_Improvement 🚀

## 🎯 项目目标（Ultimate Goal）

在6个月内，从零开始成为：

> **一个能构建完整系统 + 能将AI整合进开发流程的工程型开发者**

最终具备能力：

- 能独立构建 Fullstack 系统
- 能设计 API / 数据流 / 数据库结构
- 能使用 AI 提升开发效率（不是依赖）
- 能用 Java / Node 双路线对齐中国求职市场

---

## 🧠 最终产出（6个月后）

### 🥇 项目 1（主项目）

**AI Coding Workflow Tool**

功能：

- 输入代码 + 输入需求
- AI生成代码（refactor / explain / fix / generate）
- 展示前后对比
- 保存历史记录
- Prompt模板系统

---

### 🥈 项目 2（工程项目）

**Java Backend System**

功能：

- 用户系统
- CRUD API
- 登录认证
- MySQL数据库
- 标准分层结构（Controller / Service / Repository）

---

## 🧭 学习路线总览（6个月）

---

### 🥇 Phase 0（Week 1）

**基础启动阶段**

目标：

- 熟悉 JavaScript
- 能写基础逻辑

学习：

- function / object / array
- async / await
- Node.js 基础

输出：

- 能写简单JS代码

---

### 🥇 Phase 1（Week 2–4）

**Backend 基础**

目标：

- 能写 API
- 理解数据流

- Express / Fastify

项目：

- Code Request API（无AI版）

---

### 🥈 Phase 2（Week 5–8）

**Fullstack MVP**

目标：

- 做出完整 Web App

- TypeScript

- AI Coding Workflow Tool（初版）

功能：

- 输入代码
- 输入需求
- AI返回结果

## ⚠️ 必须遵守的原则

> 先做系统，再优化代码

### 原则 2

> AI 是辅助，不是替代

---

### 原则 3

> 每个阶段必须有“可展示成果”

---

### 原则 4

> 不做重复项目（例如Todo）
